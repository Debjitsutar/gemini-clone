#  Gemini Clone (AI Chat App)

A modern AI-powered chat application built using **React + Vite**, inspired by Google's Gemini. This app allows users to interact with an AI model and receive intelligent responses in real time.

---

##  Features

*  Real-time AI chat interaction
*  Fast performance using Vite
*  Secure API key handling using `.env`
*  Clean and responsive UI
*  Loading indicator while fetching responses

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite, CSS
* **API:** Google Generative AI (Gemini)
* **Version Control:** Git & GitHub

## ⚙️ Setup & Installation

1. Clone the repository:

```
git clone https://github.com/Debjitsutar/gemini-clone.git
```

2. Navigate to the project folder:

```
cd gemini-clone
```

3. Install dependencies:

```
npm install
```

4. Create a `.env` file in the root directory and add:

```
VITE_API_KEY=your_api_key_here
```

5. Run the development server:

```
npm run dev
```
